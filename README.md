赛博小叶子 `v1.0.0` 开发中

使用前请运行 `ollama`, 并提前下载 `qwen2.5:7b` 模型

```bash
bun install # 安装依赖
bun d # 运行
bun b # 构建
bun b:win # 在 Mac 下构建 Windows 版本
```
